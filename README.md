# Claude Code Skills

A collection of Claude Code skills for automating development workflows.

Each directory is a self-contained skill that can be used with [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Usage

Copy any skill directory into your `~/.claude/skills/` folder to use it.

## Skills

| Skill | Description |
|-------|-------------|
| [UseGit](UseGit/) | Git practices for this project. |
